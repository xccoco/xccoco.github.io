给OIER的ubuntu安装教程

2021-07-25

肝了一个晚上写出来的东西，自我觉得还是挺傻瓜式的

写在最前面

我所理解的OIER的需求就是刷刷题，听听音乐，看看视频，偶尔打打游戏。所以如果你的需求比这个要高，那么本文很可能帮不了你。但系统安装部分仍有阅读价值。

而你如果啥事没有就是想折腾，那么本文也很可能帮不了你。本文所提供的只是满足上面基本要求的教程，而不是教你怎样从零开始撘一个系统。

当然linux这个系统下，肯定是要折腾的。如果你觉得折腾一下能给自己带来更好的使用体验，那当然没问题。但你如果只是想装个13，无聊，那大可不必把时间浪费在这个上面。去刷刷题，和朋友联机打两把，哪个不比在这里折腾好？

前言

为什么我们要使用linux？

最现实的理由是：NOI官方，以及大大小小的比赛，几乎都是用linux评测的。在linux下，一些在windows下被算作警告而不提示的东西会被提示成编译错误。

举个很简单的例子，像这样的两行程序：

long long n;
scanf("%d", &n);

windows下管都不会管，直接给你转换了。但linux下这会报错。

为什么我们要使用ubuntu？

1: noi 比赛时使用的 noi linux2.0 是 ubuntu20.04 的魔改版。以前那个 noi linux1.0 也是 ubuntu14 的魔改版。

2:尽管 Arch 系系统有号称软件最全面的 AUR ，但使用中出的错误实在太多，而且绝大部分官方都提供了对应 ubuntu 的 deb 安装包，比AUR可靠。

3:ubuntu 这个系统，虽说肯定有缺点，但是linux家族中最好入门的了（ deepin 的 bug 跟不要钱一样的满天飞）。而且，相对地配置也比较简单

系统安装

这篇教程会教你将ubuntu安装进移动硬盘中，并可以实现即插即用

硬件

你需要一个U盘，保险起见64GB。但我更推荐你搞个120GB的固态硬盘，最便宜的大厂200出头可以搞到一块（我的硬盘就是搞活动199买的）

虚拟机

这里推荐使用VMware。直接去网上找个 15 或 16 的下载就可以了。这东西的盗版现在也是满天飞了。

装好后按这个流程走就行：

然后打开虚拟机，插入硬盘。应该有提示让你选择该硬盘是连接到主机还是虚拟机。选虚拟机。

如果没有，尝试着在上方“虚拟机”中点击“可移动设备”，看看你的U盘有没有在里面。

如果还是没有，请按下 Win+R ，输入 services.msc ，在其中找到 Vmware USB Arbitration Service 这一项，双击将其打开，在“启动类型”中

选择“自动”，并点击下方的启动。然后再次尝试。

如果还是没有，看看你是不是把 UBS3.0 的借口插到2.0里面了。在我的电脑上这样在虚拟机和之后选择系统时都是铁定识别不出的。不知道大众情况怎么样。

USB3.0接口是蓝色的，2.0是黑色的

系统安装与分区

如果你已经照上面的做了（这时一定要保证硬盘连上了虚拟机！），那么找到在左侧简体中文，然后选择“安装ubuntu”。之后的键盘布局不用管，点“继续”。更新把“安装ubuntu时下载更新”勾上，原因见它的注释。上方选择“最小安装”，时间会快很多（最小安装初始只有一个火狐），而且正常安装会安装一堆用不到的软件。

安装类型选择“其他选项”，我们要在移动硬盘里安装ubuntu。

如果你上面照着我的做了，这你的硬盘应该显示在上面了，下面有个“空闲”，且如果你的硬盘没动过的话，下面的“空闲”大小应和你的移动硬盘大小差不多。

如果空闲过小，尝试清理一下移动硬盘

右键点击“空闲”，选择“添加”。然后分区按下面的来

分区类型 占用空间 挂载点
Ext4日志文件系统 20GB /
Ext4日志文件系统 300MB /boot
Ext4日志文件系统 剩余所有 /home
swap交换分区 8GB

格式化应该是默认搞好了的。如果没搞好，前三个格式化，swap交换分区不格式化。

下面启动引导器设备选你的移动硬盘。例如若移动硬盘在里面显示为sdb，则你选择的设备应该就是sdb，而不是sdb1。

时区选择上海，用户名和密码自己填。然后开始安装即可！

安装可能要花费一些时间。这些时间，你最好不要用电脑干其他的事情。但你也可以另开一台电脑。。。

软件

软件安装与卸载

目前搜狗输入法、百度网盘、金山WPS、网易云音乐、VScode都提供deb安装包。除了搜狗，其他的下载完双击安装即可。

搜狗需要额外的配置，但官方已经有教程了（就在点击下载后的界面），所以这里不用再讲了。

还有，企鹅公司果然是国企之光，拿了个致远星发来的QQ版本当QQlinux版给我们用。。。然后还意图搞死第三方QQ软件

如果不想这么用，那么上百度搜索教程即可。在这里就不讲了

爷要上B站！

ubuntu本来是没有安装视频解码包的。所以要安装一下。

可以使用 ctrl+shift+t 打开终端，输入这样一条命令：sudo apt-get install ubuntu-restricted-extras

中间要按tab+enter进行确定

如果要播放在线视频，下载VLC即可。百度有官网。

爷要装13！

你可以将下载好的deb包拖到主目录，或者在终端中输入cd+deb包路径名使终端跳到该目录下，然后在终端中输入：sudo dpkg -i package.deb即可进行安装。其中package.deb改成你的deb包名即可。看起来高大上很多呢。。。

yysy，这种方法比双击安装要快，因为少了打开应用商店这一步。。。

顺便说一句，终端中让你输密码时光标本来就是不会动的！不用再调了！

软件不好用，爷要卸载！

你可以在终端输入dpkg --list后找到你要卸载的软件包，使用sudo apt-get remove 包名进行卸载即可。

你也可以在软件中心搜索Synaptic安装新利得软件管理器，支持搜索要删除的软件。但这样删除时就要经历右键，标记以便彻底删除，应用，确认四个步骤。

vscode的安装与配置

noi linux2.0已经支持vscode了。而且零配置vscode搭配终端好像比零配置vim搭配终端好用一些。至少，她不用配置，就可以进行自动重加载文件。。。

别的教程洛谷已经有日报了。这里就在说一下，要想在ubuntu下用快捷键编译运行程序，你只需要安装两个插件即可：c/c++（微软的那个）和code runner。之后，按ctrl+alt+n就可以编译运行了~

美化

系统美化

如果你下载的是ubuntu，而不是kubuntu等一众魔改版，那么你的图形化界面应该是基于gnome的。

进这个网站，按照他提示的来安装浏览器插件，在搜索user themes，点进去，点击由off变为on。然后再进应用程序里搜索，就可以搜到一个叫做“优化”或“调教”或“tweak”的东西了。里面就是有关于外观的设置。

然后还是刚刚那个网址，搜索dash to dock，安装。然后自己改里面的设置就行了。

终端美化

ubuntu默认终端实在有点丑。。。

于是下载xfce4终端：在终端中输入命令：sudo apt-get install xfce4-terminal

然后更改默认终端，运行命令：sudo update-alternatives --config x-terminal-emulator

然后看哪个是xfce，输入对应编号即可。

xfce4的扩展功能就多多了。例如乱搞个背景图什么的。
