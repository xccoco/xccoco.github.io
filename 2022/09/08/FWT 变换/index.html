FWTFWTFWT 的定义

模仿 FFTFFTFFT，对于 AAA 我们想要在可接受时间内得到一个 FWT(A)FWT(A)FWT(A)，使得

FWT(C)i=FWT(A)i×FWT(B)iFWT(C)_i=FWT(A)_i\times FWT(B)_i FWT(C)i​=FWT(A)i​×FWT(B)i​

iii 代表第 iii 个位置的数。

这样我们就可以 O(n)O(n)O(n) 暴力乘起来了。

因此我们构造 FWT(A)i=∑j∣i=iAjFWT(A)_i=\sum^{}_{j|i=i}A_jFWT(A)i​=∑j∣i=i​Aj​。

现在我们来证明这个构造的正确性。

FWT(A)i×FWT(B)iFWT(A)_i\times FWT(B)_i FWT(A)i​×FWT(B)i​

=∑j∣i=iAj×∑k∣i=iBk=\sum^{}_{j|i=i}A_j\times \sum^{}_{k|i=i}B_k =j∣i=i∑​Aj​×k∣i=i∑​Bk​

=∑j∣i=i∑k∣i=iAjBk=\sum^{}_{j|i=i}\sum^{}_{k|i=i}A_j B_k =j∣i=i∑​k∣i=i∑​Aj​Bk​

因为可以从 j∣i=ij|i=ij∣i=i 与 k∣i=ik|i=ik∣i=i 中得出 (j∣k)∣i=i(j|k)|i=i(j∣k)∣i=i，所以我们还可以消去另一个式子

=∑(j∣k)∣i=iAjBk=\sum^{}_{(j|k)|i=i}A_j B_k =(j∣k)∣i=i∑​Aj​Bk​

根据 FWTFWTFWT 的定义，这个式子就是 FWT(C)iFWT(C)_iFWT(C)i​。证毕。

另一种理解

涉及到了或运算，我们不妨把数全都变成二进制。于是我们想到一种经典转换：将二进制中的 000 与 111 转化为集合中一个数选还是不选。那么或操作代表什么呢？

bingo！两个集合的并集！

于是我们可以把上面的式子改写一个形式：

Ci=∑i=k∣jAkBjC_i=\sum^{i=k \mid j} A_kB_j Ci​=∑i=k∣j​Ak​Bj​

变成

Ci=∑i=k⋃jAkBjC_i=\sum^{i =k\bigcup j} A_kB_j Ci​=∑i=k⋃j​Ak​Bj​

注意看,这时 i,j,ki,j,ki,j,k 都是集合,只不过我们将其用二进制表示.

这样我们可以改写 FWTFWTFWT 的定义。

重新来一遍，FWT(A)i=∑k⊆iAkFWT(A)_i=\sum^{}_{k \subseteq i} A_kFWT(A)i​=∑k⊆i​Ak​。

因此我们为 FWTFWTFWT 找到了新的定义，他代表着集合 iii 的子集之和。我们有了更自然的推导：

∑j⊆iAj×∑k⊆iBk\sum^{}_{j \subseteq i}A_j\times \sum^{}_{k \subseteq i}B_k j⊆i∑​Aj​×k⊆i∑​Bk​

=∑j,k⊆iAj×Bk=\sum^{}_{j,k \subseteq i}A_j\times B_k =j,k⊆i∑​Aj​×Bk​

=∑x⊆i∑j⋃k=xAj×Bk=\sum^{}_{x \subseteq i}\sum^{}_{j\bigcup k = x }A_j\times B_k =x⊆i∑​j⋃k=x∑​Aj​×Bk​

=∑x⊆iCx=\sum^{}_{x \subseteq i}C_x =x⊆i∑​Cx​

=FWT(c)x=FWT(c)_x =FWT(c)x​

换句话说，我们对子集做了个前缀和操作（发现了吗？子集的前缀和进行或卷积与普通前缀和进行加法卷积具有相似性），并用前缀和相乘代替了原来的 O(n2)O(n^2)O(n2) 相乘。

如何变化

我们把原序列 AAA 按下标最高位是 000 还是 111 分成两部分 A0A_0A0​ 与 A1A_1A1​ 分治求解。显然，前半部分（最高位为 000 的部分）就是 FWT(A0)FWT(A_0)FWT(A0​)，所以我们考虑后半部分的答案。

后半部分最高位为 111，因此此时“子集”这一概念不仅包含分治处理的他子集，还包括把最大值变为 000 后的，序列 A0A0A0 中同一位置的子集。要将 A0A_0A0​ 中的同一位置加到当前答案上。

写成数学形式就是:

FWT(A)=merge(FWT(A0),FWT(A0)+FWT(A1))FWT(A) = merge(FWT(A_0),FWT(A_0)+FWT(A_1)) FWT(A)=merge(FWT(A0​),FWT(A0​)+FWT(A1​))

上面的 mergemergemerge 代表拼接，就是字面意思。

于是我们就能写出分治递归代码了！但为了常数着想，我们试着把递归这一步骤去掉。

去掉的部分并不难写，我们按照层数从小到大递归，不难发现第 iii 层（从 000 开始编号，最底层为 000）就是枚举第 iii 位是 000 还是 111，并且乱填其他数进行转移。

如何转回来

再看一眼转移的式子

FWT(A)=merge(FWT(A0),FWT(A0)+FWT(A1))FWT(A) = merge(FWT(A_0),FWT(A_0)+FWT(A_1)) FWT(A)=merge(FWT(A0​),FWT(A0​)+FWT(A1​))

思考只有两个数的情况。此时 111 位置是不会变的，222 位置加上了 111 位置的贡献，要减去。

我们发现更大的情况也是一样的，只要依次把前面的贡献减去就好。

这两份代码显然是可以合并的。因此我们得到了 FWTFWTFWT 或 的全过程。

正着转化

思考 FWTFWTFWT 的作用，我们想要把 AkBjA_kB_jAk​Bj​ 变成 AiBiA_iB_iAi​Bi​ 的形式，以此来简化运算。

我们考虑这样 nnn 个 nnn 维向量 bbb，b(i)b(i)b(i) 只有下标 iii 处是 111，其他位置都是 000。

现在我们把 FWTFWTFWT 后的 A,BA,BA,B 看作系数，此时显然 A1b(1),A2b(2),...,Anb(n)=A1,A2,A3,A4...,AnA_1b(1),A_2b(2),...,A_nb(n)=A_1,A_2,A_3,A_4...,A_nA1​b(1),A2​b(2),...,An​b(n)=A1​,A2​,A3​,A4​...,An​

显然，异或卷积对于乘法有分配律。

设异或卷积为 ∗\ast∗，则

(∑i=1nAib(i))∗(∑i=1nBjb(j))(\sum^{n}_{i=1} A_ib(i)) \ast (\sum^{n}_{i=1} B_jb(j)) (i=1∑n​Ai​b(i))∗(i=1∑n​Bj​b(j))

=(∑i=1n∑j=1nAiBj(b(i)∗b(j))=(\sum^{n}_{i=1}\sum^{n}_{j=1} A_iB_j (b(i)\ast b(j)) =(i=1∑n​j=1∑n​Ai​Bj​(b(i)∗b(j))

发现后面的东西可以简单表示，即 bi∗bi=bi,bi∗bj=0(i≠j)b_i \ast b_i = b_i,b_i \ast b_j = 0(i \neq j)bi​∗bi​=bi​,bi​∗bj​=0(i=j)。

∑i=1nAiBi\sum^{n}_{i=1} A_iB_i i=1∑n​Ai​Bi​

太长不看版：异或 FWTFWTFWT 与原序列线性相关

既然这样，我们设 FWT(A)x=∑i=1ng(x,i)AiFWT(A)_x=\sum^{n}_{i=1} g(x,i)A_iFWT(A)x​=∑i=1n​g(x,i)Ai​。

那么因为 FWT(C)x=FWT(A)x×FWT(b)xFWT(C)_x=FWT(A)_x\times FWT(b)_xFWT(C)x​=FWT(A)x​×FWT(b)x​

所以 ∑k=1ng(x,k)Ck=∑i=1ng(x,i)Ai×∑j=1ng(x,j)Bj\sum^{n}_{k=1} g(x,k)C_k=\sum^{n}_{i=1} g(x,i)A_i \times \sum^{n}_{j=1} g(x,j)B_j∑k=1n​g(x,k)Ck​=∑i=1n​g(x,i)Ai​×∑j=1n​g(x,j)Bj​。

整理一下可以得出：

∑k=1ng(x,k)Ck=∑i=1n∑j=1ng(x,i)g(x,j)×AiBj\sum^{n}_{k=1} g(x,k)C_k=\sum^{n}_{i=1}\sum^{n}_{j=1} g(x,i)g(x,j)\times A_iB_j k=1∑n​g(x,k)Ck​=i=1∑n​j=1∑n​g(x,i)g(x,j)×Ai​Bj​

把 CkC_kCk​ 用 A,BA,BA,B 表示可得：

∑k=1ng(x,k)∑k=i⊕jAiBj=∑i=1n∑j=1ng(x,i)g(x,j)×AiBj\sum^{n}_{k=1} g(x,k)\sum^{k=i \oplus j} A_iB_j=\sum^{n}_{i=1}\sum^{n}_{j=1} g(x,i)g(x,j)\times A_iB_j k=1∑n​g(x,k)∑k=i⊕j​Ai​Bj​=i=1∑n​j=1∑n​g(x,i)g(x,j)×Ai​Bj​

更改求和顺序，我们枚举 i,ji,ji,j 可得：

∑i=1n∑j=1ng(x,i⊕j)AiBj=∑i=1n∑j=1ng(x,i)g(x,j)×AiBj\sum^{n}_{i=1}\sum^{n}_{j=1} g(x,i \oplus j) A_iB_j=\sum^{n}_{i=1}\sum^{n}_{j=1} g(x,i)g(x,j)\times A_iB_j i=1∑n​j=1∑n​g(x,i⊕j)Ai​Bj​=i=1∑n​j=1∑n​g(x,i)g(x,j)×Ai​Bj​

于是我们发现了 ggg 的关系：

g(x,i⊕j)=g(x,i)g(x,j)g(x,i \oplus j) = g(x,i)g(x,j) g(x,i⊕j)=g(x,i)g(x,j)

现在问题来了，与 i,ji,ji,j 相关的什么东西，使异或之后的值等于原来两值的乘积？

于是我们可以想到有人托梦给我奇偶性。

具体的，我们发现异或前后 111 的个数奇偶性不变。原因如下：

按每一位依次考虑。如果第 iii 位异或后为 111，那么原来必定有且仅有一个 111。个数不变

如果为 000，要么是两个 000，此时 111 的个数不变，要么是两个 111，此时 111 的个数减 222，奇偶性仍不变。

所以我们定义 g(x,i)=(−1)∣i⋂x∣g(x,i)=(-1)^{|i \bigcap x|}g(x,i)=(−1)∣i⋂x∣。那么上式就等价于：

(−1)∣(i⊕j)⋂x∣=(−1)∣i⋂x∣(−1)∣j⋂x∣(-1)^{|(i \oplus j) \bigcap x|} = (-1)^{|i \bigcap x|}(-1)^{|j \bigcap x|} (−1)∣(i⊕j)⋂x∣=(−1)∣i⋂x∣(−1)∣j⋂x∣

根据上面的推论，左右两边奇偶性不变，与 后无非是减去两个相同的数，奇偶性还是不变。

于是我们得出 FWTFWTFWT 的转移式：

FWT(A)x=∑i=1n(−1)∣i⋂x∣AiFWT(A)_x=\sum^{n}_{i=1} (-1)^{|i \bigcap x|}A_i FWT(A)x​=i=1∑n​(−1)∣i⋂x∣Ai​

如何求解

考虑模仿前两个 FWTFWTFWT 的形式，讨论最高位 iii 为 000 和为 111 两种情况。

原来最高位为 000，FWTFWTFWT 后的前 2i−12^{i-1}2i−1 个数最高位还是 000。由于 1&0=01 \And 0=01&0=0，所以后 2i−12^{i-1}2i−1 个数的贡献为正。前半部分答案为 FWT(A0)+FWT(A1)FWT(A_0)+FWT(A_1)FWT(A0​)+FWT(A1​)。

FWTFWTFWT 后的后 2i−12^{i-1}2i−1 个数最高位变成了 111，此时 A0A_0A0​ 的贡献还是正（因为 1&0=01 \And 0=01&0=0）。但是此时后半部分加了 111，于是贡献要取反。后半部分答案为 FWT(A0)−FWT(A1)FWT(A_0)-FWT(A_1)FWT(A0​)−FWT(A1​)。

所以我们得出：

FWT(A)=merge(FWT(A0)+FWT(A1),FWT(A0)−FWT(A1))FWT(A) = merge(FWT(A_0)+FWT(A_1),FWT(A_0)-FWT(A_1)) FWT(A)=merge(FWT(A0​)+FWT(A1​),FWT(A0​)−FWT(A1​))
